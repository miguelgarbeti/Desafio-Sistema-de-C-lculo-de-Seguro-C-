##🚀Nessa atividade uzando situações hipoteticas sobre uma situaão hipotetica:

##🖥️Contexto:
Você assumiu o cargo de desenvolvedor em uma seguradora e recebeu a tarefa de implementar as novas regras de negócio para o cálculo das apólices. O sistema deve ser preciso e capaz de lidar com diferentes categorias, perfis de condutores e planos de cobertura.

##📖Como deve ser executado: 
O programa deve imprimir um relatório detalhado contendo:
Valor Base
Valor do Adicional de Categoria
Valor do Adicional de Idade
Valor da Cobertura do Plano
Valor Total Final da Apólice
